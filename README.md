# BlendShape Copy Tool - VPM Repository

VRChatアバターのBlendShape値を効率的にコピーするためのUnityエディタ拡張ツール「BlendShape Copy Tool」のVPMリポジトリです。

## 📥 インストール方法 (ALCOM / VCC)

このリポジトリをALCOMまたはVRChat Creator Companion (VCC) に追加することで、簡単にツールを導入・更新できます。

1. **リポジトリURLをコピー**:
   ```
   https://github.com/tomotos141/blendshape_copy_tool/raw/master/index.json
   ```
2. **ALCOMへ追加**:
   - ALCOMの設定（またはRepositories）画面で、上記のURLを追加します。
3. **プロジェクトへ導入**:
   - 各プロジェクトのパッケージ一覧から `BlendShape Copy Tool` を選択してインストールします。

## ✨ 特徴

- **Direct Copy**: 同一プロジェクト内のメッシュ間で値を同期。
- **JSON Export/Import**: JSONファイルを経由して、別のプロジェクトへ値を持ち出し可能。
- **Smart Matching**: 名前が一致するBlendShapeを自動的に抽出。
- **Undo対応**: 操作を間違えても `Ctrl+Z` で戻せます。
